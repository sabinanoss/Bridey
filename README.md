# bridey-landing-page
Using Sass in landing page project: global variables, architecting CSS, managing media queries, CSS grid, etc.


## Dev stack
1. HTML 5
2. CSS 3
3. Node-Sass

- CSS grid layout

# Project setup
```
npm install
```
## Compiles and live-reloads for development
```
npm run start
```
## Compiles, prefix, concat and minifies for production
```
npm run build:css
```
